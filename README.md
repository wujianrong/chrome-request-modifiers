## 主要功能
修改web页面发起的请求返回的出参内容   


## 参考项目
参考YGYOOO这位大佬的项目修改（https://github.com/YGYOOO/ajax-interceptor）  进行优化。该项目原本使用React开发，使用React的同学可以参考原项目。

## 改动功能
- 项目从react改成vue
- chrome开发V2版本升级V3版本
- UI交互调整
- json编辑器更换
- 增加搜索功能
- 支持修改404、502等失败类型的返回体


## 安装
0. 拉取源码
1. npm install
2. npm run build
3. 谷歌浏览器 - chrome://extensions/ - 加载已解的拓展程序 - 选择项目源码目录
